Инструкция по работе с программой:

Первым делом установи используемые библиотеки (openpyxl, numpy, matplotlib), иначе программа
просто не запустится и выдаст ошибку. Потом измени путь к Эксель таблице, из которой будут браться
данные. Если нужно измени название листа, где находятся данные, по умолчанию название "Sheet1".

Далее в функции main() введи нужные столбцы со значениями, проставь обозначения для
оси координат, названия графиков, используемый метод обработки (массивы x, y, x_label, y_label, tit, ind).
Также исправь номер ячейки, с которой начинают браться данные, их количество и нужное количество графиков
(массивы start, data и значение переменной numb). Соответствующим образом подкорректируй цикл.

Запусти программу. Проверь, что все графики построены верно. Далее введи обозначения для каждой линии
(массив label) и раскомментируй строчку, отвечающую за сохранение файла с графиком.
